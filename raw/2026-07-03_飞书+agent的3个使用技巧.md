# 飞书 + agent 的 3 个使用技巧

## 让收藏夹不再落灰，把它跟日历打通

做一个浏览器插件，把"稍后阅读"清单跟飞书日历打通。

- 插件代码：https://github.com/zarazhangrui/reading-block-lark
- 制作全过程：https://bytedance.aiforce.cloud/app/app_1799jpnkj29/

## 用 HTML 做动态看板/数据可视化

### （财务场景）预算沙盘
创业团队资金规划，支持预算滑块调节、收入增长模拟、融资开关、多情景切换。

### （销售场景）报价配置器
套餐选择、席位调配、增值模块勾选、折扣拉取，实时出报价单并计算毛利，毛利低于红线变红警告。

### （增长/营销场景）增长漏斗分析
营销预算→访客→注册→激活→付费的完整漏斗，支持CAC、LTV、LTV/CAC计算和回本月数。

### （招聘场景）薪酬 offer 计算器
拆解基本年薪、奖金、签字费、公积金、股权等，退出倍数滑块直观展示offer上下限。

## 飞书现已支持上传 HTML！

上传HTML到云空间可直接分享给同事、把控权限；也可以部署到飞书妙搭获得可分享网页链接。轻量单页用云空间，复杂前端效果用妙搭。

## 在 Codex 客户端侧边栏直接打开飞书文档

在 Codex 电脑端 app 里写飞书文档时，加一句"在侧边栏打开"即可，无需跳转浏览器。
