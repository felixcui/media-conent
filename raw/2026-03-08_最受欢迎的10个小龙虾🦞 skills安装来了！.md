#  Datawhale干货 

**作者：王熠明、刘伟鸿，Datawhale成员**

最近科技圈的“小龙虾”特别火——也就是 OpenClaw。![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/vI9nYe94fsEa8rC0gTqdaU15zYvOBA7p201ID94YHu6xDibTsXC2NUd4Nw5T06RFxcxZ2daXpJicicHt7pQUZXeBw/640?wx_fmt=png&from=appmsg&wxfrom=5&wx_lazy=1&tp=webp#imgIndex=0)OpenClaw 的核心优势是“技能包（Skills）”系统——它就像一个万能工具箱，AI 遇到什么任务就掏什么工具，专具专用，高效不卡顿。

目前社区里已经有几百个这样的“工具”了，直接挑花了眼。上周，我们专门为新手筛选出了 **[10 个最建议安装的 Skills](https://mp.weixin.qq.com/s?__biz=MzIyNjM2MzQyNg==&mid=2247719882&idx=1&sn=bb0e4da8922fc2418172041a84375f86&scene=21#wechat_redirect)。今天我们来点进阶干货。******

**通过X上大家讨论和推荐的实用/高频技能（基于社区反馈、安装量、使用场景，排除明显恶意类），我整理出了下面这些被多次提到“很推荐”“必装”“日常用”的，基本覆盖核心需求：**

**1. tavily-search**

联网搜索技能（AI优化版），让Agent能实时查最新资讯、数据，避免“闭眼编”。几乎所有人都说“没这个跟瞎子一样”。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/zW6S9vt0cS9LKLorO8VlU3ZNIdUCm4U5iaKaq5HeqopUXl9OCybMjGZp6CkF3fQRNpvMQghW3V5ydUuM2YHt2qiaJgnUTqbJ2SzibOp1ZJ1teY/640?wx_fmt=png&from=appmsg#imgIndex=1)安装：npx clawhub@latest install tavily-search

**2. find-skills**

让Agent自己去ClawHub搜并安装需要的技能，解决“不知道用哪个工具”的痛点。 

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/zW6S9vt0cSicnGFJL7V27tfEP0psSEfBQj2peEtvFmzxkQbu57z4nUr9nsENdwKM5RfbRJjDvhtQgKfEOPF25YEC5VF1gDCGqjHCCh4lZmMM/640?wx_fmt=png&from=appmsg#imgIndex=2)安装：npx clawhub@latest install find-skills

**3. proactive-agent**

（或旧版proactive-agent-1-2-4） 给Agent加“主动性”和自我迭代能力，能记住历史、优化行为、减少重复问。长期用很香。

![图片](https://mmbiz.qpic.cn/mmbiz_png/zW6S9vt0cS9RNrv4UIEaOZicJE8e0xEB2SoKcJDLtk9Uia8nLrJMEHLuj73Ilzh22IL4pPKVn3hVgQRelNCJheaicKwYyTibmkU5Huwhbm5lsmg/640?wx_fmt=png&from=appmsg#imgIndex=3) 安装：clawhub install proactive-agent

**4. github**

GitHub集成，能自动管repo、issue、PR、code search。开发者必备。

![图片](https://mmbiz.qpic.cn/mmbiz_png/zW6S9vt0cSico63dUMEH4mXtn5qN1gkFare0g6NiaVDwiaXfYeOrhducKZpJYMITKpjjEGdqaA71YiakQJA885RicWZhHjDUAPclaf7B9ZTaoWPU/640?wx_fmt=png&from=appmsg#imgIndex=4) 安装：npx clawhub@latest install github

**5. gog**

Google Workspace全家桶（Gmail、日历、Drive、Docs），办公自动化神器。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/zW6S9vt0cSicyutPjdVrhsGTovtKJIF45zhicqefXBvQaCsicftCMgIh2ibRnRDHljWFiacqzpZfHUqdB4IqEO5qxsPSIrX6JuWE5PMm5UObvCibY/640?wx_fmt=png&from=appmsg#imgIndex=5)安装：npx clawhub@latest install gog

**6. skill-vetter** / **clawsec**

安装前扫描技能安全，防恶意代码。**安全第一步**，很多人后悔没先装这个。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/zW6S9vt0cSicNmRCoxIumFpUQvAgXes2kCZvzlrL9VdnNLdZPibpL5Ks8ic15fADSIuTIVaqZoOficaS3NcrwvpTPVibv63PFibIrzIf26z091ouk/640?wx_fmt=png&from=appmsg#imgIndex=6) 安装：npx clawhub@latest install skill-vetter 或类似安全扫描类

**7. automation-workflows**

工作流编排，把多个技能串起来干复杂事（自动邮件+报告+数据同步等）。

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/zW6S9vt0cSibs6dnRrjibE7gXmddVVw3hFYcn7a36fCjne0BaVxo3u0vn4CCfHHOuIDEX0ehWRibvkgLI3T04EK47diatcvJbRmu3v2YnGfq5Zo/640?wx_fmt=png&from=appmsg#imgIndex=7) 安装：npx clawhub@latest install automation-workflows

**8. bird**

![图片](https://mmbiz.qpic.cn/mmbiz_png/zW6S9vt0cS9zD078MmVSj8gP58c4RF60p2ibEgFG2MQG0oZ5sGp27JhdD1NXlwK8CmHkKFBjsqVBkzocbmew39jmHN5GlDicsba2aKuwsBUhQ/640?wx_fmt=png&from=appmsg#imgIndex=8)X/Twitter集成，能发帖、搜趋势、管理账号（但小心伪装恶意版）。（社区提到，但慎用高赞的“Twitter”相关）

**9. self-improving-agent** 

![图片](https://mmbiz.qpic.cn/sz_mmbiz_png/zW6S9vt0cS84PI8ny6ylqHuKRia5fnyz1IV7Aib7GdIqpaYhSnAV9Rv5LxJCf5z40yN5CHzbTrgibUDAJJydKxmjnep3t0X3M8UeFAXKAUmNiaA/640?wx_fmt=png&from=appmsg#imgIndex=9)加记忆+自我优化，长期交互越用越聪明。 

安装：npx clawhub@latest install self-improving-agent

**10. feishu-doc**（针对国内用户） 飞书文档/云盘集成，企业/团队协作常用。

![图片](https://mmbiz.qpic.cn/mmbiz_png/zW6S9vt0cSibJWftrRFxOqHBUsXOZ5KzTvDImWcich9MZN294TbXKGOyzpmnAcoNIHQpeItMBK8AE3gicgqAInjY9iaFRRwHsC8wodkQSNqmS98/640?wx_fmt=png&from=appmsg#imgIndex=10)安装：npx clawhub@latest install feishu-doc

![图片](https://mmbiz.qpic.cn/mmbiz_png/vI9nYe94fsGxu3P5YibTO899okS0X9WaLmQCtia4U8Eu1xWCz9t8Qtq9PH6T1bTcxibiaCIkGzAxpeRkRFYqibVmwSw/640?wx_fmt=other&wxfrom=5&wx_lazy=1&wx_co=1&tp=webp#imgIndex=2)**
