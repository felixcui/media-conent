最近，我们看到了一位用户的真实使用方式： **他把扣子接进了自己的日常 A 股交易辅助流程里，** 盘前整理市场信息，盘中监控持仓变化，盘后复盘交易策略，再把每天的执行结果继续带回系统中迭代，收益获得大幅提升。

对他来说，扣子是一个能持续理解上下文、调度任务、帮他提取投资信息的“助手”，每天协助他完成交易。下面，我们就来分享一下他的使用方法～



> 注：本文仅分享此用户将扣子用于信息整理、任务调度与复盘辅助的真实使用方式，不构成投资建议。所有金融相关内容仅作为信息整理和辅助分析参考，最终判断仍需由用户自己完成。



![](https://feishu.cn/file/VzvObLVILokb9Gxlv8YcEsfenPb)

这位用户原本在互联网做产品运营，并不是金融科班出身。一开始，他想解决的问题很直接：自己在财务、金融投资上并不是专业人士，很多判断靠直觉并不稳定，能不能借助 AI，用更低成本补上那部分缺失的判断能力？



他开始用扣子来尝试补上这个“短板”。刚开始，他零星地问扣子一些个股、策略和判断，但很快他发现，交易辅助不是一个“一问一答”的问题。今天怎么判断，往往和昨天的复盘、当前持仓、风险偏好、历史策略和盘中变化都有关。

**于是他长期和扣子持续沟通，扣子的长期记忆会记住他的目标和偏好，并辅助搭建出一套协作流程。**



他用扣子搭建出的这套交易辅助系统，思路很清晰：扣子负责承接长期上下文和用户目标，同时给扣子设定定时任务，每天检查运行状态；同时他还接入了云服务器和自动化脚本，用来定时扫描市场信息、检查持仓变化，并处理行情、公告等数据，最后扣子整理汇总，把关键结果推送给他。

![](https://feishu.cn/file/H2UebmdLFoPQ4UxjDcTcnFOknch)

整个决策系统的搭建，其实并不复杂，直接跟扣子对话就能完成。

**他通过对话设定目标，Agent 会去搜索相关信息并提供解决方案。** 由于扣子 Agent 配置有云电脑，可以像人一样“操作”云端电脑，提供的解决方案也会更全面。在过程中有任何问题，都可以直接让扣子 Agent 自行解决，他只需要进行判断并下达指令。



在这套交易辅助系统中，他给扣子的角色定位像是一个“调度员”：不只是回答一句话，而是理解目标、拆解任务、连接工具、安排执行，再把结果整理清楚发给他，做到能看懂、能使用。

![](https://feishu.cn/file/TDxObRfhCo43FBxmfZ3cQO1rn0c)

当这套系统真正跑起来，扣子就不再是“想起来才问一下”的工具，而是嵌进了一个完整交易日，每天执行固定的任务。扣子 Agent 支持设置定时任务，指定任务内容、执行时间后，Agent 就能按照要求自动执行，在日历中也能管理 Agent 每天的“工作日程”。



他给扣子设定的循环任务是这样： 在开盘前汇总信息、盘中实时巡检、盘后根据结果复盘，并为下一轮提供建议。

![](https://feishu.cn/file/EYGgbnFyHortkbximkIcSGypnMY)

扣子每天的日程

- 开盘前：关键信息整理汇总

前一晚外围市场有什么变化，持仓相关公司有没有公告，新闻里有哪些线索，重点主题有没有新的动向，这些都会进入盘前整理。像半导体设备、半导体材料、封测产能这类主题，也被他配置成固定追踪任务。



扣子会把这些信息和之前的复盘结果结合起来，形成盘前情报卡和当天的默认操作策略。这一步，能把分散的信息先整理成一个可以行动的判断底稿。让他在开盘之前不用从零开始翻资料，而是先拿到一份被组织过的“今日准备”。

![](https://feishu.cn/file/QcJvbYbkYoP5prxIA3ecNzebnlb)

- 盘中：按照节奏持续巡检

他给扣子设定的关注不是单个问题，而是一组会随市场变化不断更新的条件：持仓资金变化、止损止盈、异动信号、V 反转、反弹机会、建仓条件。

![](https://feishu.cn/file/EUIVbq2xTowwSTxEvtTcuGOQnQf)

重要信号，会通过扣子电脑端、手机App 推送给他，同时他也把扣子渠道配置到微信，扣子也会通过微信来提醒他。对于交易这种强时效任务来说，及时非常重要，所以扣子多端、多渠道的支持会让信息提醒更方便。

![](https://feishu.cn/file/EaJAbqY3IoqpTlxxFuZc6X8mnde)

- 收盘后：复盘优化

一天的交易后，还会有持续复盘。当天有没有严格执行策略，哪里卖早了，哪里止损慢了，哪个判断过于保守，哪些规则第二天需要调整，都会重新回到对话里。



这位用户会把真实结果带回来，和扣子一起讨论有哪里没做好，当天的结果，会继续变成下一天的提醒、规则和计划。同时在晚间继续巡检信息，这些结果都放自动整理在第二天的盘前信息中。

![](https://feishu.cn/file/VznAbKlTQoavT5xu09RcqDjKnEe)

推到扣子端的盘后公告扫描和盘后选股

![](https://feishu.cn/file/O8Hubbwzao4OX0xTP3Tc8LaAnqd)

- 提供专业信息支持

对这位用户来说，扣子最直接的价值，是帮自己在不够了解的地方，多了一层信息整理、逻辑校验和规则提醒。



之前投资可能不清楚市场变化情况，现在扣子先把信息整理清楚，把风险规则摆出来，把复盘逻辑接上，所有信息一目了然，再由他自己做最后判断。



- 持续优化策略

在整个过程中，有几个很有意思的细节。比如5月底一次和扣子协作的过程中，账户收益一路高涨，但后来因为没有按策略卖出，导致收益回撤。



在复盘中，这样的操作也会纳入总结里，他会让扣子继续补规则、补脚本、补数据源、补提醒条件，持续优化这套系统。

![](https://feishu.cn/file/XECEbODCPojYpaxKngRc31q4nQb)

- 提供更多决策参考



除了投资信息，在日常其他决策中，扣子也能给出有效的建议。

比如在筛选数据源的时候，他本来在考虑购买一个月大概2000的数据源，但扣子结合目前已经有的数据库，建议只需要买一个月200左右的数据库，就已经够用了，能补足数据信息。



这也是长期深度使用之后的另一个价值：更了解背景信息之后，同时也能提供更针对性的建议。
