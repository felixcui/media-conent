# 腾讯云Agent Memory节省61% Token提升52%成功率的诀窍：Mermaid无限画布×上下文卸载

**作者**: 腾讯云开发者
**来源**: 微信公众号
**日期**: 2026-05-26
**链接**: https://mp.weixin.qq.com/s/kDR0amCQl502R_0eGZJFBw

本文提出一种基于上下文卸载与结构化图表示的短期记忆压缩方案：通过将完整信息卸载至外部文件系统，同时利用 Mermaid 图语言将任务执行过程转化为可导航的结构化记忆，使完整信息保留于外部、关键状态以高密度形式留存于上下文。在超长Session实验中，该方案最高节省 61% Token，同时把超长Session任务的通过率从 33% 提升到 50%，相对提升52%。从实验结果可以看出，短期记忆压缩能力尤其适合办公提效、创作、研究和编程这类长任务和多任务场景，该能力已随 TencentDB Agent Memory 产品化节奏发布。

⭐Github仓库地址：https://github.com/Tencent/TencentDB-Agent-Memory。

引言：什么是"一语胜千言"？

我们每天都在做一件很厉害的事：用一句话概括一整段经历。

但 Agent 还不会主动这样解析输入。当它执行一个长任务时，会不断搜索、读文件、调用工具、修改代码。每一步都在产生大量信息。如果这些内容全部堆进上下文，Token 被迅速耗尽，模型被细节淹没，甚至逐渐偏离原本的目标。

TencentDB Agent Memory 通过上下文卸载 + Mermaid 无限画布，把长任务过程转化为可管理的结构化记忆。在超长 Session 实验中，该方案最高节省 61.38% Token，同时把任务通过率最多提升 51.52%。

01 压缩的艺术

1.1 日常生活中的语言压缩
语言的一个基本能力，是把复杂经验压缩成可传播的符号。词语本身就是一种压缩装置。"家"并不只是一个居住空间，"江湖"也不只是地理意义上的江与湖。语言压缩并不依赖字面信息的完整保留，而依赖语义结构的有效保留。

1.2 极致压缩：大模型能理解什么样的"符号"
压缩的本质：用更少的 token 表达等量（或近似等量）的语义。大模型不仅能理解自然语言，还能高效理解"符号"，如 MBTI 人格类型。但压缩不能依赖模型"记住了什么符号"，而应依赖模型"能够从符号中推理出什么结构"。符号设计三条原则：必须是通用知识、生成不能过于复杂或孤立、表达要足够自由。

02 "符号化压缩"与无限画布

2.1 从"压缩单条信息"到"组织任务结构"
长任务压缩不能只停留在"把内容压短"，还要把压缩后的信息组织成结构。线性列表像日志，图结构像地图。

2.2 什么是 Mermaid 图
Mermaid 是一种用纯文本描述图结构的标记语言，适合 Agent 读写。

2.3 为什么 Mermaid 适合作为任务画布
Mermaid 把离散 summary 组织成一张任务拓扑，表达"这些信息之间是什么关系"。Mermaid 符合符号化设计三原则：通用知识、简单语法、自由表达。

2.4 载体选型：Flowchart vs StateDiagram
Flowchart 作为压缩载体效果比 StateDiagram 好约 15%。Flowchart 更适合描述 Agent 高度自由的探索式执行过程。

2.5-2.8 从图到画布
无限画布的本质：不是让上下文窗口无限变大，而是让上下文之外的信息仍然可见、可定位、可恢复、可继续执行。Agent 采用层次化注意力机制：鸟瞰 → 聚焦 → 下钻。

03 工程实践

3.1 什么是上下文卸载
上下文卸载就是把 Agent 的工作记忆变轻：眼前只放当前要用的信息，细节放在外部，靠索引随时找回。

3.2 为什么单独用上下文卸载或无限画布都不够
上下文卸载解决"信息太长"，但不解决"结构丢失"；Mermaid 画布解决"结构"，但信息可能压缩太狠。两者结合：细节可恢复，结构不丢失。

3.3 记忆的折叠
一次工具调用结束后，信息拆成三种形态：完整结果写入 refs/*.md，工具调用压缩成 JSONL 记录，任务状态更新到 Mermaid 画布。

3.4 记忆的展开
四层分级：refs（完整原文）→ JSONL（工具摘要）→ MMD（任务节点）→ metadata（任务入口）。

04 实验

4.1 实验设计：超长 Session 压力测试，50个 task 串在同一个 session 中连续执行。

4.2 总体结果：Token 消耗明显下降，任务完成率没有下降反而提升。

4.3 SWEbench：完成率从 58.4% 提升到 64.2%，Token 节省 31%-33%。

4.4 Toolathlon：通过率从 20% 提升到 30%-35%。

4.5 WideSearch：Token 节省最高 63.59%，通过率从 8.5% 提升到 12%。

4.6 AA-LCR：准确率从 44.0% 提升到 47.5%，Token 节省约 31%。

4.7 消融实验：仅有上下文卸载时 Token 节省约 15%，完整方案达 31%-33%，证明 MMD 画布贡献显著。

05 产品化与开源

短期记忆解决"怎么集中注意力完成当前任务"，长期记忆解决"下一次再见面时 Agent 怎么更懂你"。产品已上线 Qclaw、Lighthouse、ClawPro 等，即将 GitHub 开源。
