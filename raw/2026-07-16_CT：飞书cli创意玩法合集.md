## 飞书cli安装

项目地址：

```Markdown
https://github.com/larksuite/cli
```



安装cli 

```Markdown
npm install -g @larksuite/cli
```

配置 claude code 使用

```Markdown
npx skills add https://github.com/larksuite/cli.git -y -g
```

初始化及授权

```Markdown
lark-cli config init
```
