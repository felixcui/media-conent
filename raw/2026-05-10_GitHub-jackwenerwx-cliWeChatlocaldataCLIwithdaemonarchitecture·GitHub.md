## 摘要

wx-cli 是一个基于 Rust 构建的命令行工具，用于在本地查询微信数据，支持会话、聊天记录、联系人、群成员、收藏及统计导出等功能。该工具采用 daemon 架构持久缓存解密数据库，实现毫秒级响应，且数据完全本地处理，保障隐私安全。它提供默认 YAML 及可选 JSON 的 AI 友好输出格式，可通过 npm、curl 或手动下载等方式零依赖安装，并支持一键集成到 Claude Code 等。
