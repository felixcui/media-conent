# 别画了！用飞书 CLI 真能一句话生成架构图

**作者**: 银海

**来源**: https://mp.weixin.qq.com/s/kGZS9pjaiAAfhqaGEIIiJg

---

## 摘要

飞书CLI推出了一项新功能，可通过AI Agent（如Codex）实现一句话生成架构图。与传统繁琐的画图工具不同，该功能直接生成可编辑的飞书在线画板，节点和连线均可随时双击修改或拖拽调整。画板无缝嵌入飞书文档中，避免多窗口切换，且需求变更时可通过与AI对话快速迭代更新，彻底解决了传统画图耗时长、修改成本高的痛点，大幅提升了工作效率。

我想你们可能都经历过这种事：写方案写到一半，要补一张架构图，打开画板工具，拖节点，对齐，调字号，调颜色。

四十分钟过去了，拿给老板一看："这块逻辑改一下。"行，全部重来。

需求文档里画泳道图更离谱，拖到手酸，对齐对到眼瞎。好不容易画完截图贴进文档，第二天需求变了，又得从头搞。

![](https://mmbiz.qpic.cn/mmbiz_png/thoHNWXYDzTeTcz4KN9NtwSuiclPpK1mClv7MicR2uSZ3icPBTP7Vza7Pt1Fy4OeeM1GDGtJX4Mj3xI5MMuynlJicKnkH8Ne2Viacq7uLLjib00Mg/640?wx_fmt=png&from=appmsg)

飞书 CLI 这个技能不一样，它出来的是飞书画板本身，能够直接给我们的 AI Agent（Codex、Claude Code、OpenClaw等等）直接集成使用。

像我一开始的这个图就是直接在 Codex 里面通过这么简单的一句话给生成的，然后会返回给我一个飞书画板的在线链接。

节点、连线、文字都能改，不满意就双击进去调，不用打开别的工具。

需求变了也简单，挪个节点改个字，几秒钟的事。而且画板直接嵌在飞书文档里，不用来回切窗口，文档沉淀了画板也跟着沉淀，下次复用直接改。

![](https://mmbiz.qpic.cn/mmbiz_png/thoHNWXYDzQNCzpe8hia9seq9m5Dibib0rlLHGN5618M8PtMXalO52JujrjBical7iapVm9sHwotzicgHia1EHIptBwkXUZ3Bxm33aL3tZR4kAujqI/640?wx_fmt=png&from=appmsg)

当然了，如果想要再修改这个画板，除了手动去编辑细节以外，我们也可以直接跟 Codex 这种AI Agent直接去对话就好：“这个图能给改成中英文双语版本吗”

它能够快速的帮我们修改好，并且一样更新到飞书在线画板里面。

![](https://mmbiz.qpic.cn/mmbiz_png/thoHNWXYDzRHHCd9G8icxEosCIVicqrTsa128jjhiaPSu96M5oEBCQyRlr3diabEoxfiaRrjzabAbYBUXppuxnvwzXH04sl7kAibL7vYQiaEibC5wlc/640?wx_fmt=png&from=appmsg)

产品经理梳理功能模块，脑子里大概知道有哪些模块、怎么交互，但要画成产品架构图给团队对齐又得折腾半天。

现在直接一句话："画一张电商后台的产品架构图，包含商品管理、订单管理、用户管理、营销活动、数据看板。"

画板直接出来，模块怎么分组、层级怎么排都给你搞好了，如果要加模块？双击进去补。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/thoHNWXYDzTrOczm6yygXdYhPyDf1mMzsqwvnAaH6CvvWYLokBxNc2p4D4a89IAmAAicHCv7Ugu6pQM5a27QSYRQGnbg03k4ibd9efyNnfAew/640?wx_fmt=png&from=appmsg)

那么这个飞书 CLI 怎么装呢？有两种方法。

方式一：手动安装。在终端中输入一行命令完成安装

```
npx @larksuite/cli@latest install
```

方式二：将以下信息发送给你的 AI Agent 工具（如 Codex、Claude Code）

```
帮我安装飞书 CLI：https://open.feishu.cn/document/no_class/mcp-archive/feishu-cli-installation-guide.md
```

装好就能用。

当然了，直接在飞书的 aily 里面直接就是内置好了，左侧打开就能用。

![](https://mmbiz.qpic.cn/mmbiz_gif/thoHNWXYDzRH2k2ltsdA8LADlFnfK7gY3FFk7eXD1wr6dqn9zQCbFZwJx0ibaNJCF50nN4jiaYvK3okBfXwMib1ynJZaVQt3tPRLibdRpH6fR94/640?wx_fmt=gif&from=appmsg)

我自己在使用飞书 CLI 画板的过程中，为这个飞书画板美化还做了一个风格 Skill，配合这个飞书 CLI 画板能力使用也非常好用，已经开源在 Github 上了，如果你有需要也可以直接让你的 AI Agent 安装上。

![](https://mmbiz.qpic.cn/mmbiz_gif/thoHNWXYDzR2VyY8DryUAL27etlibPygnDM9wYWickGM7a5wWfplfQ1GE2MsbLSqYrzowJIPIk6GK9MEh6N8I5NvIKw9j17Th9MicOVTF0g7WE/640?wx_fmt=gif&from=appmsg)

地址是：https://github.com/inhai-wiki/feishu-whiteboard-themes

默认生成的画板样式不一定适合所有场景，技术架构图可能想要科技感强一点的，产品 roadmap 可能想要简洁清爽的，给老板汇报可能想要正式一点的。

这个风格 Skill 会根据你画的图的类型，自动推荐合适的视觉风格，配色、排版、组件样式都会跟着变。

以前用 AI 画图，出来的图改不了或者改起来比重画还麻烦。

这次不一样，生成的画板就是飞书画板本身，能改能复用，直接嵌在文档里。

画图这件事，终于不用再当体力活干了。
