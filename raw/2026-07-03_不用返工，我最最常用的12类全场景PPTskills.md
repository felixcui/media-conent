# 不用返工，我最最常用的12类全场景PPT skills

**作者**: Sherry小水

**来源**: https://mp.weixin.qq.com/s/jqW4lhVvDbhnywKqIk2nNg

---

## 摘要

作者分享了利用Codex制作高质量PPT的解决方案，通过将HTML设计模板库整理成12套全场景PPT技能，彻底解决了传统AI生成PPT文字模糊、难修改且缺乏动效的痛点。这些模板分为注重设计感的个人展示类和正式的职场汇报类，能直接生成兼具设计感与动态效果的HTML版PPT，实现无需返工的高效制作。

***效果展示***

这里小水先来给大家展示下这12套的效果，当然它们的名字也都非常好听。

总共分两大类，

一类是偏向自媒体、个人展示、作品集这类更注重设计感的模板，

另一类是偏职场汇报、政务行政、产品演讲这种更正式的场合。

我们先从第一类开始。

第一套，粉彩拼贴志。

这套我自己最常拿来做教程复盘，

比如有一个流程要讲清楚，或者要完整展示一个项目的过程，那它真的非常适合。

![](https://mmbiz.qpic.cn/mmbiz_gif/KIiaW5rrtFCWfibbicWdsrd3m8BrQ8b1diaWP4sad85zqxNu1I6tAbD0DSXGp8Xl9gR5QlJxHCKcIoZlUPULRd65pBliaiaicBTXdh4NfIbiaqyN33E/640?wx_fmt=gif&from=appmsg)

因为整个视觉体验是一种放在桌上手帐本的感觉。

米白色纸底、粗粗的黑色描边，加上几块粉彩色块。

看起来虽然不像一份正经PPT，但呈现出的内容往往显得很有设计感。

![](https://mmbiz.qpic.cn/mmbiz_png/KIiaW5rrtFCUFNY69InBFbJvaADibW3Ms8icF3HLZJvUglhbUV8E0DLNJdQIxpUZVooynUujic1RSWUdpGcmz0RJ30N8wibn31AsV7a6K9DsibVsY/640?wx_fmt=png&from=appmsg)

看到它，会觉得这个人是真的花心思在做的。

第二套，暖粉编辑志。

这套我其实更推荐给有品牌感需求的内容创作者，

![](https://mmbiz.qpic.cn/mmbiz_gif/KIiaW5rrtFCXIkzPYVIJxDXjwsBz1bylLicsqD3efjf4Nxv1Bpiad5M1lD6RRiaias7F4BugJ5NYeBzpfXqZZzFBNA0AbRksJqZCfh4XD3nkAmLs/640?wx_fmt=gif&from=appmsg)

整体就有一种小红书图文的感觉。

而且亮粉色一跳出来，观众的视线会自动落在最想让人看到的信息上。

第三套，墨线白稿。

这套是这一组里最效果干净简洁的，

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/KIiaW5rrtFCUnMaloAmFLI3v8ibCZfQoXSw9nWrU83c1pMVsjbGkDZ4JRyjY6B3q0xZ53AcfXJcO0BxLTBjibP4PRLibowMOfqvMwgNND0spK2s/640?wx_fmt=gif&from=appmsg)

手写风格的标题加上曲线装饰，整体质感是草稿本上尚处于概念阶段的呈现。

特别适合内容还在打磨、 但又想拿出来讲一讲的场景。

虽然给人一种看起来没做完，但效果往往更偏真实。

***使用教程***

这里文字展示下，安装提示词以及具体的使用方法，

非常非常简单易懂（看不懂你来找我！）

![](https://mmbiz.qpic.cn/mmbiz_png/KIiaW5rrtFCU3vDcwOz5thD1ksdsZYuAkqFClexLG7F74POGAo47ABtSwXZXHTmpPPJJrmlUClfia1SkP3RxY0TJ8D4JjHGhfMuoNt0X1fnoc/640?wx_fmt=png&from=appmsg)

首先你得有一个codex，以及一个能够登录Openai的账号。

当然Claude code或者龙虾这类的Coding Agent也都可以。

这里我们就以我常用的codex举例，因为整套ppt skill我已经上传到了github。

所以我们可以直接输入网页地址，并告诉codex进行安装，详细的提示词我放在下面。

1.安装飞书 CLI

```
帮我安装飞书 CLI：https://open.feishu.cn/document/no_class/mcp-archive/feishu-cli-installation-guide.md
```

这一步是为了把飞书上面的文档和Agent连接上。

这样子Agent就可以直接调用飞书知识库里，你希望拿来做ppt的文档初始信息。

2.安装xiaoshui-pretty-ppt 的skill

```
安装这个skill，https://github.com/XshuiAi/xiaoshui-pretty-ppt.git，
```

这一步是为了安装，Agent可复用的ppt制作逻辑。

那安装完毕后，重启codex，就可以使用了。

具体用法可以参考下列提示词。

```
从飞书链接生成 HTML 网页 PPT

使用 $shui-pretty-ppt，把这个飞书文档做成一份适合对外分享的 HTML 网页 PPT。

要求：

1. 先判断内容更适合哪一套模板；

2. 把长文档拆成封面、钩子、核心观点、案例、流程、总结；

3. 开启可编辑模式，方便我录视频时展示生成后还能改；

4. 最后告诉我本地 HTML 路径和如何打开。
```

当然，这是让AI根据内容自动进行选择。

你也可以根据我提前设计了12套不同的网页ppt模板，提前确定好要用的模板。

比如像这样，可以替换里面的模板名称（Cobalt Executive Deck/钴蓝商策）

```
指定模板生成

使用 $shui-pretty-ppt 的 Cobalt Executive Deck / 钴蓝商策，

把这份产品介绍做成商务汇报型 HTML 网页 PPT。

请开启可编辑模式。
```

当你做好后，直接全屏投屏展示就可以了。

如果你需要插入图片的话，可以直接告诉他哪一页需要修改就好了。

如果你想改任何东西，直接跟AI说他就可以直接帮你修改。
