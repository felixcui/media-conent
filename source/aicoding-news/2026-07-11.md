## 编程模型

[**Claude Code 团队官方：如何设计 Agent Loop？**](https://mp.weixin.qq.com/s/_NqXsu7dx_LkZSBmZMFqRw)
Claude Code 团队指出，使用 coding agent 的重点正从写提示词转向设计 Agent Loop，即 Agent 反复执行工作直到满足停止条件的循环过程。设计 Loop 需关注触发方式、停止条件及任务类型等维度，应从简单解法起步。

[**Loop 是什么？怎么用？**](https://mp.weixin.qq.com/s/odSibIpZ7fcYuxeK_xIA7A)
Loop是AI Agent重复执行工作循环直到满足停止条件的过程，本质是将任务反复交给大模型自动执行。随着Agent的发展，使用AI的核心已从写提示词转向“设计Loop”，即规划让AI在循环中承担多少工作。目前主要有四种Loop模式，例如传统的回合式Loop仍需人类充当“人肉编译器”进行人工验收，而更高级的目标式Loop则通过/goal指令将验收标准量化交给AI，实现不达标不停止的自动化验证与修复。
