
相关工具：hermes，飞书, github ，defuddle，lark-cli，macmini

hermes 部署在家里 macmini 上。

通过飞书发送需要收集的 url给 hermes，hermes 执行 news_collect 技能
技能主要功能如下：
	获取公众号及链接内容，保存 markdown 格式到本地git 仓库。公众号及普通文章使用defuddle，飞书文章使用lark-cli
	写入 url，标题，摘要到飞书多维表格。使用 lark-cli。
	写入 ima 知识库， 使用 ima-skills。
	写入 notebooklm，使用qiaomu-anything-to-notebooklm。
