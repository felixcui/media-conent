

**说明：**本文档基于 2026 年 6 月可检索的公开资料整理。视频创作类产品迭代很快，功能、价格、模型接入、商用条款和数据政策都可能变化。正式采购或接入前，仍应以官网、合同、隐私政策和版权条款为准。

# **一、总体介绍

这 6 个产品都可以归入“视频创作 Agent”或“Agent 化视频生产平台”的范畴，但它们解决的问题并不一样。简单看，Topview 和小云雀更偏营销短视频与电商内容；TapNow 和 LibTV 更偏画布、节点、多模型编排和工作流；OiiOii 更偏动画故事；Buzzy 更偏自然语言视频编辑和视频精修。

· 要快速做广告、带货、产品短视频，可以优先看 Topview、小云雀、Buzzy。

· 要搭建可控的视频生产工作流，或让 Agent 调用视频能力，可以重点看 LibTV、TapNow。

· 要让非专业人员快速做动画故事，可以关注 OiiOii。

# **三、重点产品说明**

# **1. Topview**

**产品地址：**[官网](https://www.topview.ai/)（https://www.topview.ai/）

**定位：**面向营销、电商和社媒广告的 AI 视频 Agent。

Topview 更像一条“营销视频生产线”。它不是先要求用户懂剪辑，而是从商品链接、产品图、参考视频或一句话需求出发，自动生成适合社交平台投放的短视频。对于电商、品牌广告和本地生活服务来说，它的价值在于出片快、变体多，能支持反复测试不同标题、卖点和口播风格。

## **核心功能**

· AI Video Agent / Smart Canvas：用自然语言描述需求，生成产品广告、社媒短视频、迷你短片等。

· URL to Video / Product Video：输入商品链接或上传产品图，自动提取卖点并生成宣传视频。

· UGC / Avatar：用 AI 数字人、口播、类真人推荐方式生成带货或测评视频。

· 广告模板与渠道适配：按 TikTok、Instagram Reels、YouTube Shorts 等平台尺寸与风格生成内容。

· 参考复刻与多版本生成：上传参考视频或广告风格，生成多个方向用于 A/B 测试。

## **典型应用场景**

· 跨境电商商品广告、DTC 品牌广告、Amazon/TikTok 商品视频。

· 本地生活服务介绍，如餐厅、教育培训、门店宣传。

· 社媒矩阵号批量生成短视频素材。

· 投放团队快速生成不同 Hook、节奏、口播版本。

## **优势**

· 商业化导向明确，适合快速出广告素材。

· 模板和渠道分类清晰，非专业剪辑人员也比较容易上手。

· 适合批量生成变体，提高广告投放测试效率。

## **局限与注意事项**

· 内容风格容易广告化、模板化，需要结合品牌调性做二次调整。

· 复杂叙事、严肃专题、长片控制能力有限。

· 涉及商品素材、人物形象、参考视频复刻时，需要确认版权和肖像授权。

# **2. TapNow**

**产品地址：**[官网](https://www.tapnow.ai/)（https://www.tapnow.ai/）；[应用入口](https://app.tapnow.ai/)（https://app.tapnow.ai/）

**定位：**面向创作者和企业的 Agentic Creative Canvas。

TapNow 的气质更接近“AI 版创意工作台”。它把文本、图片、视频、音频和模型调用放在同一个画布里，通过节点连接出一条可复用的创作流程。它不是简单地让用户输入一句提示词等结果，而是让创作者可以一边生成、一边拆解、一边改镜头和故事板，适合做更复杂、更可控的视觉项目。

## **核心功能**

· Tapflow Canvas：通过节点连接文本、图片、视频、音频，形成可复用创作流程。

· 多模型集成：在一个工作区调用不同图像/视频模型，减少跨工具切换。

· Script-to-Video / Image-to-Video：脚本成片、图生视频、分镜生成。

· Cinema Lab：提供摄像机运动、光照、构图、空间控制等偏电影化能力。

· TapTV 社区：可以分享、复刻、拆解、重混工作流。

## **典型应用场景**

· 广告片、概念短片、AI 短剧、实验影像。

· 创意团队做分镜、风格探索和镜头迭代。

· 需要多模型协同的视觉项目，例如先生成角色图，再生成镜头，再生成音频。

· 团队协作式项目：导演、设计、剪辑、运营共同维护一套节点流程。

## **优势**

· 节点式工作流可控性强，适合复杂项目。

· 可把创作流程沉淀为“配方”，后续复用和改版更方便。

· 适合探索 AI 电影感、镜头语言和长故事分段创作。

## **局限与注意事项**

· 学习成本高于“一句话成片”工具。

· 复杂视频生成会消耗更多额度和等待时间。

· 节点流程更适合专业团队，普通运营人员刚开始可能觉得偏重。

# **3. OiiOii**

**产品地址：**[官网](https://www.oiioii.ai/)（https://www.oiioii.ai/）；[体验入口](https://app-hoe.hogi.ai/)（https://app-hoe.hogi.ai/）

**定位：**对话式 AI 动画制作 Agent。

OiiOii 主打的是“把动画团队放进聊天窗口”。用户不用一开始就准备完整脚本，也不用理解复杂的模型参数，而是像和一个动画制作小组沟通一样，逐步确定角色、剧情、分镜和画面风格。它尤其适合从一个模糊想法出发，快速做出动画故事的雏形。

## **核心功能**

· 对话式动画制作：像和 ChatGPT 聊天一样提出动画需求。

· 多 AI 工作角色协同：围绕角色设定、脚本、分镜、镜头生成自动推进。

· 故事模式：从故事概念到角色表、分镜、视频片段逐步生成。

· 可编辑分镜与重生成：生成后可对镜头进行局部调整和重新生成。

· 调用外部图像/视频模型：公开体验资料显示其支持多种图像和视频模型。

## **典型应用场景**

· 动画短片、儿童故事、科普动画、教学动画。

· 社媒上的风格化动画广告、CM 风动画。

· 不懂 Prompt 和模型选择的新手做 AI 动画入门。

· 教育培训机构生成趣味故事视频。

## **优势**

· 对新手友好，不需要理解复杂模型和节点。

· 从“想法”到“动画”路径比较清晰，能减少空白页压力。

· 适合快速做故事原型和动画样片。

## **局限与注意事项**

· 自由度和专业控制力不如节点画布型工具。

· 对高级创作者而言，固定流程可能显得不够灵活。

· 产品仍在快速变化，应重点关注商用权、数据存储和模型调用条款。

# **4. 小云雀**

**产品地址：**[App Store](https://apps.apple.com/cn/app/%E5%B0%8F%E4%BA%91%E9%9B%80-%E6%8A%A2%E5%85%88%E4%BD%93%E9%AA%8Cseedance2-0/id6746231056)（https://apps.apple.com/cn/app/%E5%B0%8F%E4%BA%91%E9%9B%80-%E6%8A%A2%E5%85%88%E4%BD%93%E9%AA%8Cseedance2-0/id6746231056）；[Microsoft Store](https://apps.microsoft.com/detail/xpdc2tsl7b7r7d)（https://apps.microsoft.com/detail/xpdc2tsl7b7r7d）；[剪映官网](https://www.capcut.cn/)（https://www.capcut.cn/）

**定位：**剪映/抖音生态的视频与图片 AI 创作助手。

小云雀的优势在于“离中文短视频生态很近”。它把视频成片、数字人口播、AI 图片设计和换背景放在一起，让不会剪辑、不会设计的人也能先做出一个可用初稿。对于新媒体日更、电商卖点视频、课程宣传和活动预热，它更像一个轻量化的内容助理。

## **核心功能**

· 智能生视频 / 智能成片：输入主题或素材，自动生成短视频。

· 数字人口播：输入文字，调用数字人形象生成讲解视频。

· AI 图片设计：一句话生成风格化图片、海报、商品图。

· AI 换背景：抠图、光影优化、背景替换，适合商品图和宣传图。

· 照片会说话 / 一镜到底：让照片口播，或基于图片生成连续运镜视频。

## **典型应用场景**

· 短视频账号日常内容生产。

· 电商商品图、卖点视频、促销海报。

· 课程宣传、知识口播、活动预热。

· 不方便真人出镜时，用数字人完成标准化讲解。

## **优势**

· 中文场景友好，降低剪辑和设计门槛。

· 贴近抖音/剪映生态，适合短视频分发。

· 数字人口播、海报、换背景形成一站式轻量营销套件。

## **局限与注意事项**

· 专业镜头控制、长片叙事和复杂项目管理能力有限。

· 生成内容仍需要人工审核事实、导向、版权和肖像风险。

· 模板化风险较高，需要结合自有素材和人工精修。

# **5. Buzzy**

**产品地址：**[官网](https://www.buzzy.now/)（https://www.buzzy.now/）

**定位：**AI Director / Vibe Video Photoshop。

Buzzy 的表达很直白：让用户像修图一样修视频。它强调通过聊天完成视频编辑和生成，比如去掉画面里的物体、调整光线、改变风格、控制镜头，尽量减少反复重拍和繁琐后期。它更适合在已有素材或已有创意基础上做快速改造，而不是只从空白提示词开始生成。

## **核心功能**

· 自然语言视频编辑：通过对话完成裁剪、重风格化、加效果、场景细化。

· Storyboard Creator：生成详细故事板，尽量维持角色、物体、地点等跨场景一致性。

· Multi-angle Camera Control：通过空间控制调整镜头角度和运动。

· Relighting：调整主光、亮度、色温、轮廓光等。

· 多模型接入：公开页面展示其接入多种主流视频模型。

## **典型应用场景**

· 对已有视频进行快速二创、改镜头、改风格。

· 短视频团队复刻爆款结构，生成多版本。

· 广告团队做视频素材优化和视觉包装。

· 创作者做故事短片、产品片、社媒视频。

## **优势**

· 更接近“视频修图/视频精修”思路，适合对已有素材做快速修改。

· 故事板、镜头、灯光控制有助于提高成片一致性。

· 多模型整合减少在多个工具之间来回切换。

## **局限与注意事项**

· 上传素材、参考视频、人物视频时需特别审核服务条款和授权范围。

· 自然语言编辑结果仍可能不稳定，复杂修改需要多轮迭代。

· 如果用于爆款复刻，要避免侵犯他人版权、肖像权和平台规则。

# **6. LibTV**

**产品地址：**[官网](https://www.liblib.tv/)（https://www.liblib.tv/）；[GitHub Skill](https://github.com/libtv-labs/libtv-skills)（https://github.com/libtv-labs/libtv-skills）

**定位：**面向人类创作者和 AI Agent 的专业 AI 视频创作平台。

LibTV 的特点不是某一个生成按钮，而是“人和 Agent 都能用”。人可以在画布里做创作，Agent 可以通过 OpenAPI/Skill 创建会话、上传素材、发送生成或编辑指令、查询进度并获取结果。对于想把“文章转短视频”“脚本转分镜”“专题转宣传片”做成内部流程的机构来说，它比单点视频生成工具更值得关注。

## **核心功能**

· 生成能力：文生图、文生视频、图生视频、视频续写。

· 编辑能力：局部修改、元素替换、镜头调整、风格迁移。

· 复杂创作：一句话生成短剧、复刻视频风格做 TVC/宣传片、音乐生成 MV、产品展示片。

· Agent Skill / OpenAPI：创建会话、发消息、上传文件、查询进展、下载结果。

· 模型编排：公开 Skill 文档提到可通过 LibTV 调用生图、生视频等 AIGC 能力。

## **典型应用场景**

· 企业内部“文章转短视频”“脚本转分镜”“专题转宣传片”流程。

· Agent 自动生成视频初稿，人工在画布中复核修改。

· 短剧、MV、宣传片、产品片的半自动化生产。

· 技术团队将 AI 视频能力接入内容管理系统、知识库或审核系统。

## **优势**

· API/Skill 友好，适合被 Agent 调用。

· 画布与自动化结合，兼顾人工控制和机器编排。

· 更适合做机构内部视频生产链路的底座候选。

## **局限与注意事项**

· 需重点评估企业级权限、安全、审计、稳定性和 SLA。

· 复杂任务可能需要轮询和较长生成时间。

· 涉及外部模型调用时，需要确认数据出境、内容训练、版权归属和合规条款。

# **四、选型建议**

如果只是看“哪家生成的视频更好看”，很容易选偏。视频创作 Agent 真正落到业务里，关键是能不能嵌入内容生产流程，能不能被审核，能不能复用模板，能不能和素材库、版权库、发布系统连接起来。

· 轻量运营层：小云雀、Topview，适合快速生成宣传短视频、口播、海报和广告素材。

· 创意制作层：TapNow、Buzzy，适合视频创意、镜头设计、故事板、视频精修和多版本探索。

· 系统集成层：LibTV，适合 Agent/API 集成，把“文章-脚本-分镜-成片”接入内部内容生产链路。

· 动画表达层：OiiOii，适合动画故事、科普、课程、儿童化表达和原型验证。

# **五、可参考的视频创作 Agent 流程**

|   |   |   |   |
|---|---|---|---|
|**阶段**|**Agent 任务**|**可用产品**|**人工控制点**|
|选题|根据栏目、热点、历史内容生成选题和标题|通用大模型 + 内部知识库|政治导向、事实准确性|
|脚本|生成 30 秒、60 秒、3 分钟版本脚本|通用大模型 + 小云雀 / Topview|口径、敏感表述、引用来源|
|分镜|拆成镜头、画面、字幕、配音稿|TapNow / LibTV / OiiOii|人物、场景、视觉风格|
|生成|生成画面、视频、数字人口播|Topview / 小云雀 / Buzzy / LibTV|版权、肖像、素材授权|
|剪辑|自动字幕、配音、转场、封面|剪映 / Buzzy / LibTV|节奏、字幕、审美统一|
|审核发布|导出发布素材并进入人工终审|内部审核系统 + 发布系统|导向、版权、平台规范|


# **六、结论**

如果要做“视频创作 Agent”，不应把它理解成某一个神奇工具，而应理解成一套组合系统：选题 Agent、脚本 Agent、分镜 Agent、素材 Agent、生成/剪辑 Agent 和审核 Agent 共同工作。Topview、小云雀、Buzzy 更适合解决内容生产效率问题；TapNow、LibTV 更适合沉淀工作流和系统能力；OiiOii 则适合把动画故事这类创意内容做轻量化。
