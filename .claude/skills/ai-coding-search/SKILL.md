# AI Coding 资讯搜索

从 weekly 目录下的资讯文件中搜索指定主题的相关内容。

## 数据源

资讯文件位于项目的 `weekly/` 目录下：
- `weekly/2025_all.md` - 2025年全年汇总
- `weekly/2026-01-*.md` - 2026年每周资讯

## 搜索流程

1. **解析用户查询**：提取用户想搜索的主题关键词
2. **读取资讯文件**：使用 Grep 工具在 weekly 目录搜索关键词
3. **提取匹配条目**：找到包含关键词的资讯标题和摘要
4. **格式化输出**：按相关度展示搜索结果

## 执行步骤

### Step 1: 搜索关键词

使用 Grep 工具在 weekly 目录搜索：

```
Grep pattern="<关键词>" path="<project_root>/weekly" output_mode="content" -C=5
```

### Step 2: 解析搜索结果

资讯格式为：
```markdown
### [标题](链接)

摘要内容...
```

从搜索结果中提取：
- 资讯标题（### 开头的行，包含 `[标题](链接)` 格式）
- 原文链接（标题中的 URL）
- 摘要描述（标题后的段落）

**重要**：必须完整提取标题文本，不要省略或截断。

### Step 3: 输出格式

**必须严格按照以下格式输出搜索结果**：

```markdown
## 搜索结果：「<关键词>」

找到 N 条相关资讯：

---

### 1. 完整标题文本

- **链接**：https://mp.weixin.qq.com/s/xxxxx（完整URL，不要截断）
- **时间**：2026年1月31日

> 摘要内容...

---

### 2. 完整标题文本

- **链接**：https://mp.weixin.qq.com/s/xxxxx（完整URL，不要截断）
- **时间**：2026年1月24日

> 摘要内容...

---
```

**格式要求**：
- **标题**：必须完整输出，不要省略或截断，不要用 Markdown 链接格式包裹
- **链接**：必须单独一行输出完整 URL，确保可复制使用
- **时间**：从以下来源提取（优先级从高到低）：
  1. 摘要开头的日期格式（如 "2025年11月24日"）
  2. 文件名中的日期（如 `2026-01-24.md` → "2026年1月24日"）
- 如果摘要开头有日期，不要在摘要中重复显示
- 每条资讯之间用 `---` 分隔
- 摘要使用 `>` 引用格式
- 按时间倒序排列（最新的在前）
- **禁止**：不要把标题和链接合并成 `[标题](链接)` 格式，要分开显示

## 使用示例

用户输入：
- "搜索 Skills 相关资讯"
- "查找 MCP 的文章"
- "有哪些关于 Cursor 的内容"
- "ai-coding-search Claude Code"

## 注意事项

- 支持模糊匹配，关键词不区分大小写
- 如果搜索结果过多，优先展示最新的资讯（2026年优先）
- 可以使用多个关键词进行精确搜索
- 如果没有找到结果，建议用户尝试相关同义词
