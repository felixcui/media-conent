如何用 Codex 画框架图

写技术文章、做方案汇报时，先让 Codex 把结构画出来，人再判断重点、删减细节、调整表达。

关键不是替代审美，而是快速得到一张能改、能讨论、能沉淀的初稿。
一句需求 → 结构初稿
資料 + 目標 + Codex = editable first draft
