# Codex Framework Diagram Video

基于 `output/wechat/如何用 Codex 画框架图.md` 生成的 Remotion 竖版短视频。

- Composition: `CodexFrameworkDiagramVideo`
- Size: 1080×1920
- FPS: 30
- Duration: 36s
- Source images: `public/article-images/`

Commands:

```bash
npm run lint
npx remotion still CodexFrameworkDiagramVideo --frame=30 --scale=0.25
npx remotion render CodexFrameworkDiagramVideo out/codex-framework-diagram-video.mp4
```
