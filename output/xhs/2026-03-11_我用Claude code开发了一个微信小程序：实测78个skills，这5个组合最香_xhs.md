# 🔥 Claude Code装了78个skills？实测后这5个组合最香！

✨ 装了一堆skills从来不打开？
✨ 不知道哪个skill适合哪个阶段？
✨ 想用Claude Code做个项目却无从下手？

宝子们，作者用Claude Code开发了一个「资产管理+人生目标」小程序，实测78个skills后，整理出这套**开发阶段组合拳**！

---

## 🎯 阶段一：原型阶段 → brainstorming

不知道产品该做什么？一团乱麻？

调用 **brainstorming** skill，让CC帮你头脑风暴！

关键技巧：问题要具体，CC会给十几道选择题，从产品定位到技术栈，像做心理测试一样选下来，产品形态就清晰了。

---

## 💻 阶段二：UI阶段 → ui-ux-pro-max + planning-with-files

**ui-ux-pro-max**：50+样式、97种配色、57种字体组合

技巧：把产品定位翻译成设计语言，告诉CC你要什么风格、要避开什么雷区，效果翻倍！

**planning-with-files**：大项目必用，帮你创建task_plan.md、findings.md、progress.md，随时问"我现在做到哪了"都能答！

---

## 🔧 阶段三：开发阶段 → test-driven-development

微信小程序调试成本高，每次改完要重新编译、真机扫码。

**先写测试用例，再写功能代码**，重构时秒级验证，不用手动输入一堆数据去试！

---

## 📦 阶段四：沉淀阶段 → 基于对话生成专属skills

项目做完了别急着exit！

发这句话：**"基于我们刚才的对话，给我整理一下我以后可能需要的skills"**

让CC帮你提炼专属的skills！

---

## 📌 5个Skills总结

| 阶段 | Skill | 用途 |
|------|-------|------|
| 原型 | brainstorming | 头脑风暴，明确方向 |
| UI | ui-ux-pro-max | 设计界面 |
| UI | planning-with-files | 规划任务 |
| 开发 | test-driven-development | 测试驱动 |
| 沉淀 | 自定义 | 提炼专属skill |
