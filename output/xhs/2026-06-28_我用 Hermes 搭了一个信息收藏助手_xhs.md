# 🔥 我用 Hermes 搭了一个信息收藏助手，收藏终于不吃灰了！

✨ 家人们，你们有没有这种感觉：文章收藏了一堆，真正要用时却怎么都找不到。
✨ 收藏很容易，复用很难；链接越存越多，知识却没有沉淀下来。
✨ 所以我给自己搭了一个信息收藏助手：看到好内容，直接发给飞书。
✨ 后面的抓取、整理、入库、同步，都交给 Hermes 自动跑！

## 🎯 这套流程怎么工作？

### 1️⃣ 飞书是入口：随手发链接就行
- 看到公众号、网页文章或飞书文档，只需要把 URL 发到飞书里。
- Hermes 收到后，会自动执行 `news_collect` 技能，不用我再复制粘贴。

### 2️⃣ 自动抓正文：先把内容拿干净
- 公众号和普通网页用 Defuddle 提取正文。
- 飞书文章通过 `lark-cli` 获取内容，减少手动整理成本。

### 3️⃣ 多处沉淀：不是只“收藏”一下
- 正文保存成 Markdown，放进本地 GitHub 仓库，长期可迁移、可搜索。
- 标题、链接、摘要写入飞书多维表格，变成内容数据库。
- 同步到 ima 知识库，方便后续搜索、问答和总结。
- 导入 NotebookLM，适合做深度消化、提纲和信息图素材。

### 4️⃣ 最后回到 Obsidian：真正变成我的资料库
- Mac mini 定期把仓库同步到远端。
- MacBook 再拉取到本地，进入 Obsidian 参考资料库。
- 写作、研究、整理选题时，可以直接引用和复用。

## 💡 我的真实感受

这套系统最有价值的地方，不是工具多炫，而是让信息从“随手收藏”变成**可检索、可整理、可复用**。

以前我总纠结一篇文章该存哪里，现在只要发给飞书就够了。
