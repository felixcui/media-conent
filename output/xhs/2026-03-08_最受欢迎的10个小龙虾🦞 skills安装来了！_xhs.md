## 🎯 10个必装Skills

### 1️⃣ tavily-search（联网搜索）
**没这个跟瞎子一样！** 让AI实时查最新资讯，避免"闭眼编"
```
npx clawhub@latest install tavily-search
```

### 2️⃣ find-skills（自动找工具）
让Agent自己搜并安装需要的技能，**解决"不知道用哪个"**
```
npx clawhub@latest install find-skills
```

### 3️⃣ proactive-agent（主动性+记忆）
给Agent加"主动性"和自我迭代能力，长期用越用越香
```
clawhub install proactive-agent
```

### 4️⃣ github（开发者必备）
自动管repo、issue、PR、代码搜索
```
npx clawhub@latest install github
```

### 5️⃣ gog（办公神器）
Google全家桶集成：Gmail、日历、Drive、Docs
```
npx clawhub@latest install gog
```

### 6️⃣ skill-vetter（安全第一步！）
**装其他前先装这个！** 扫描技能安全，防恶意代码
```
npx clawhub@latest install skill-vetter
```

### 7️⃣ automation-workflows（工作流编排）
把多个技能串起来，自动邮件+报告+数据同步
```
npx clawhub@latest install automation-workflows
```

### 8️⃣ bird（X/Twitter集成）
发帖、搜趋势、管理账号（注意辨认真假）
```
npx clawhub@latest install bird
```

### 9️⃣ self-improving-agent（越用越聪明）
加记忆+自我优化，长期交互持续进化
```
npx clawhub@latest install self-improving-agent
```

### 🔟 feishu-doc（国内用户友好）
飞书文档/云盘集成，团队协作必备
```
npx clawhub@latest install feishu-doc
```
