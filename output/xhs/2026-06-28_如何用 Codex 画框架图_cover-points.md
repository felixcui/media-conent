# 如何用 Codex 画框架图 - 封面图要点

## 主题
**如何用 Codex 画框架图**

## 副标题
一句话生成可编辑框架图｜让 AI 出初稿，你来微调

## 风格说明
- **风格**：Notion 极简风格，手绘线条，纸张质感
- **配色**：柔和温暖，不刺眼
- **尺寸**：小红书竖版图片，比例约 3:4
- **调性**：专业、可信赖
- **标题**：字体大且醒目，突出主标题视觉层级

## 封面展示要点

| # | 要点 |
|---|------|
| 1 | 痛点切入：告别手动拖拽、对齐、改文案的低效画图 |
| 2 | 两种组合方案：Codex + Excalidraw、Codex + 飞书画板 |
| 3 | Excalidraw = 个人草图，轻量自由、可继续编辑 |
| 4 | 飞书画板 = 团队协作，多人查看、编辑、沉淀 |
| 5 | 生成的不是死图，而是可编辑的图形 |
| 6 | 提示词模板：结构清晰、层级分明、文字简洁 |
| 7 | 核心价值：AI 快速出初稿，人来微调重点 |
| 8 | 选型口诀：个人用 Excalidraw，团队用飞书 |
